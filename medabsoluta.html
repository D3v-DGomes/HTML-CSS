Medidas Absolutas

Tipos de medidas absolutas

  1. px = é um pouco maior que os pixels do monitor;
  2. pt = é mais ideal para definir o tamanho das fontes (principalmente se for um site para impressões);
  3. in = utilizado para definir o tamanho em polegadas;
  4. cm e mm = para definir de acordo com a unidade de medida de cada um;

Medidas Relativas

Tipos de medidas relativas

  1. em = 1em é igual a multiplicação dele pelo item superior. A medida dele fica atrelada ao tamanho do font-size do item superior a ele (efeito cascata);
  2. rem = 1rm é igual a multiplicação dele pelo primeiro item da estrutura (html). Ideal para manter todo o site em tamanho de fonte proporcional ;
  3. vh e vw = servem para usar a proporção da tela do usuário. Ótimos para estruturar um menu.
  4. inherit = serve para herdar forçadamente as propriedades de um elemento em que ele está contido.
